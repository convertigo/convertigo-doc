##### Defines a <i>Tag</i> component. <br/>

 <p>This component acts as a standard HTML element defined with a starting and an ending tag.</p><p>All mobile components in the palette are already defined with a tag name. You can create your own mobile elements by using the <i>Tag</i> component specifying its <i>Tag name</i> property.</p><p>You can also add <i>Attr</i> components to a <i>Tag</i> component to create custom attributes to this element.</p><p>For example, you could create your own <code>&lt;ion-tab&gt;</code> element and add a <code>title</code> attribute to it.</p><p>For more information: <a href='https://www.w3schools.com/html/html_elements.asp' target='_blank'>HTML Elements</a>, <a href='https://ionicframework.com/docs/components' target='_blank'>Ionic Components</a></p>

Name | Description 
--- | ---
Comment | Describes the object comment to include in the documentation report.  <p>This property generally contains an explanation about the object.</p>
Identifier | Defines the component identifier.
Is active | Defines whether the component is active.
Is self-closing | Defines whether the HTML tag is self-closing.
Tag name | Defines the HTML element's tag name.
