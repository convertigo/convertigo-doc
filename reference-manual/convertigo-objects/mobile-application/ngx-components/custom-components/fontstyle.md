##### Defines a <i>FontStyle</i> component. <br/>

 <p>This component helps defining font CSS rules for a Text container component.</p><p>The <code>Family</code> property displays available font families of the <i>Application</i> or <i>Shared component</i> through their installed <i>Font</i> components.</p><p>The precedence depends on the <i>FontStyle</i> components ordering.</p>

Name | Description 
--- | ---
Comment | Describes the object comment to include in the documentation report.  <p>This property generally contains an explanation about the object.</p>
Family | Defines the font family.
Is active | Defines whether the component is active.
Selectors/Classes | Selectors/Classes
Size | Defines the font size.
Style | Defines the font style.
Weight | Defines the selector(s) and or class(es) for which this font style applies. e.g.: ion-text
