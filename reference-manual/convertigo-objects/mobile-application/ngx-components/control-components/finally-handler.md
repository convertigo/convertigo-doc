##### Defines an <i>Finally Handler</i>. <br/>

 <p><i>Finally Handler</i> can be used to execute some final actions once the main action chain is complete, whether there were errors or not.</p><p>To do so, place a <i>Finally Handler</i> immediately under an <i>Event</i>, an <i>AppEvent</i>, a <i>PageEvent</i>, a <i>SharedAction</i> or a <i>Subscribe Handler</i>, and add to it any child action chain.</p>
