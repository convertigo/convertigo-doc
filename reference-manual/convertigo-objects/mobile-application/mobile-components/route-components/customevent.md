##### Defines a route *CustomEvent* component.

This component helps defining an event which is emitted when any Convertigo *requestable* response is retrieved.


Property | Type | Category | Description
--- | --- | --- | ---
Comment | String | standard | Describes the object comment to include in the documentation report.<br/>This property generally contains an explanation about the object.
Is active | boolean | standard | Defines whether the event is active.
Marker | String | standard | Defines a marker for client application.
Source | String | standard | Defines the event source.
